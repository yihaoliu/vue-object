# vue-object
> 新建一个Vue项目

## 应用的开源框架
-  vue-material:  一个material风格的css ui框架
-  vue-router: vue的路由框架
-  axios:  Vue的路由框架
-  vuex:  vue的数据管理框架

## Plugin of production
所有的依赖在 package.json 可以看到

``` bash
#克隆项目
git clone git@github.com:yihaoliu/vue-object.git

# 进入项目
cd vue-object

# 安装依赖
npm install

# 运行项目
npm run dev

# 打包项目
npm run build
```
